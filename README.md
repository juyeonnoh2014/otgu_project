> 프로젝트 간략 소개 한 문장 
- 시즌별/성별/개인이 가지고 있는 옷을 어떻게 입을지 추천하는 웹서비스
## 핵심 기능  Key Feature
- 인스타그램 데이터베이스의 멀티 해시태그 조합으로 게시물을 보여줌 
- 성별 선택 가능 
- 계절 선택 가능
- 다양한 패션 아이템 선택 / Type 가능
## 사용 How To Use
- 성별 / 계절을 선택한 후, 개인이 활용하고 싶은 아이템을 선택 혹은 타입을 했을 시 인스타그램의 관련 해시태그들을 조합하여 적절한 게시물을 보여줄 수 있게 한다.
